# 智能知识管理系统 - 前端页面

## 📖 项目介绍

这是一个基于 Vue 3 开发的智能知识管理系统前端界面，提供了现代化的用户界面和丰富的交互功能。系统支持文档管理、知识库构建、聊天功能等核心特性。

## ✨ 主要特性

- 📁 **知识库管理** - 支持多层级文档分类和管理
- 💬 **智能聊天** - 集成AI聊天功能
- 📄 **文档管理** - 支持多种文档格式（PDF、Word、Markdown等）
- ⚙️ **系统配置** - 灵活的系统配置选项
- 🎨 **现代化UI** - 基于现代设计语言的用户界面
- 📱 **响应式设计** - 适配不同屏幕尺寸

## 🛠 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **开发语言**: JavaScript
- **样式**: CSS3
- **图标**: SVG图标 + 自定义图标

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖

```bash
# 克隆项目
git clone [项目地址]

# 进入项目目录
cd 14.前端页面

# 安装依赖
npm install
```

### 开发环境运行

```bash
# 启动开发服务器
npm run dev

# 项目将在 http://localhost:5173 启动
```

### 生产环境构建

```bash
# 构建生产版本
npm run build

# 构建完成后，dist 目录包含生产环境文件
```
